# Asset licenses

Every third-party asset in this repo, where it came from, and its license.
`npm run fetch-assets` downloads them; nothing lands in the repo without an
entry here.

| Asset | Location | Source | License |
| --- | --- | --- | --- |
| Andika font (Regular, Bold; latin subset) | `public/fonts/andika/` | SIL, served via [Google Fonts](https://fonts.google.com/specimen/Andika) | [SIL Open Font License 1.1](https://openfontlicense.org/) |
| Emoji glyphs (word pictures, creatures, map icons) | rendered at runtime from the device's system emoji font | Apple Color Emoji / Segoe UI Emoji / Noto Color Emoji, whichever the device provides | no distribution — nothing bundled |
| Celebration particles, buttons, backgrounds | generated at runtime (Phaser graphics) | this repo | same as project |
